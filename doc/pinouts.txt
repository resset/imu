Shell via ST-Link:
  SERIAL - USART3
    PD8 AF7 TX
    PD9 AF7 RX

Ground Control (radio):
  SIO - USART2
    PA3 AF7 RX

Servo:
  PWM - TIM3
    PA6 AF2
    PA7 AF2
    PB0 AF2
    PB1 AF2

Altimeter:
  I2C - I2C1
    PB7 AF4 SDA
    PB8 AF4 SCL

Gnss:
  SIO - UART8
    PE1 AF8 TX
    PE0 AF8 RX

Debug UART:
  SIO - UART7
    PE8 AF7 TX
    PE7 AF7 RX

IMU:
  SPI - SPI1
    PA15 AF5 NSS
    PB3  AF5 SCK
    PB4  AF5 MISO
    PB5  AF5 MOSI

LEDs:
  PAL - GPIO
    PA1  pullup
    PD10 pulldown
    PD14 pulldown
